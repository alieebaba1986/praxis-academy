1. md \tugas (membuat folder baru bernama tugas)
2. md \tugas\pr (membuat folder baru bernama tugas dan di dalam folder tugas akan dibuat folder bernama pr)
3. cd tugas (berpindah ke folder tugas, pastikan kamu berada pada drive tempat folder tugas berada)
4. cd .. (berpindah ke folder sebelumnya)
5. D: (berpindah ke drive D > ketik drive yang dituju) 
6. /a : Menampilkan semua file yang ada dalam folder, termasuk file tersembunyi, file system, read only file, dsb.
7. C:/tugas.doc F: (menyalin file tugas.doc yang berada pada drive C: ke drive F:)
8. C:/*.doc F: (menyalin semua file yang berekstensi .doc pada drive C: ke drive F:)
9. del d:/tugas.doc (menghapus file tugas.doc dalam drive D:)
10. rd musik (menghapus folder bernama musik)
11. ren tugas.doc kelompok.doc (mengganti nama file tugas.doc menjadi kelompok.doc)
12. type test.txt
13. shutdown /s /t 0 (shutdown komputer)
14. shutdown /r /t 0 (restart komputer)
15. ipconfig (Perintah ini untuk melihat alamat IP komputer anda)
16. CALL (Memanggil sebuah program batch dari program batch yang lain)
17. CD / CHDIR (Menampilkan nama atau mengubah direktori sekarang.)
18. CHCP (Menampilkan atau mengubah nomor active code page.)
19. CHKDSK (Memeriksa sebuah disket/harddisk dan menampilkan laporan status disket/harddisk tersebut.)
20. CHKNTFS (Menampilkan atau mengubah proses pemeriksaan disket pada saat booting.)
21. CLS (Menghapus layar jendela command prompt.)
